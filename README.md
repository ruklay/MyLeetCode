# LeetCode Practice Repository

This repository contains my LeetCode problems and solutions for **personal practice and learning purposes**.  

All problem descriptions are summarized in my own words using AI and the solutions are implemented in Python.  

---

## Structure

- Each problem has its own folder named like `0001-two-sum`  
- Inside each folder:
  - `README.md` contains a short summary and approach
  - `solution.py` contains the Python solution

---

## Goals

- Practice coding and problem-solving skills
- Experiment with AI-assisted code and summaries
